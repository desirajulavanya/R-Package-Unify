# List supported file types

`supported_types()` returns the file extensions currently supported by
`unfiy`.

## Usage

``` r
supported_types()
```

## Value

A character vector of supported file extensions.

## Examples

``` r
supported_types()
#> [1] "csv"  "tsv"  "xlsx" "xls"  "dta"  "sav"  "rds"  "json"
```
